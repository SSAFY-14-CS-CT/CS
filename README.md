## 📘 진행 방식

- **스터디 기간**: 총 8주
- **진행 요일**: 매주 수요일
- **주간 분량**: 약 50페이지씩
- **진행 방식**:
  1. 매주 지정된 분량 읽기 (~수요일까지)
  2. 각자 내용을 `.md` 파일로 정리
  3. 스터디 시간에 핵심 개념 공유 및 토론
  4. Q&A 및 발표자 교체
